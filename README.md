# Synergy4Life Lead Command Center

Version 1 is a mobile-first CRM for iPhone-sized screens. It includes a dashboard, lead cards, add/edit/delete lead flows, search, status and service filters, sample data, and LocalStorage persistence.

## Run locally

```bash
npm install
npm run dev
```

> This project is built with React, Vite, Tailwind CSS utility classes, and browser LocalStorage.
